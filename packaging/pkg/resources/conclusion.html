Aeromux has been installed successfully!

Prerequisite:
The library librtlsdr is required for RTL-SDR device access. Install with: brew install librtlsdr.

1. Configure Aeromux:
Edit the configuration file to set up your SDR devices, Beast TCP sources, and receiver location:
nano ~/Library/Application\ Support/aeromux/aeromux.yaml

2. Download the aircraft database (optional):
aeromux database update --config ~/Library/Application\ Support/aeromux/aeromux.yaml

3. Start the live TUI:
aeromux live --config ~/Library/Application\ Support/aeromux/aeromux.yaml

Uninstall:
sudo /opt/aeromux/bin/aeromux-uninstall
Add --purge to also remove configuration, database, and log files.

Man page:
man aeromux

Documentation and source code:
https://github.com/nandortoth/aeromux
